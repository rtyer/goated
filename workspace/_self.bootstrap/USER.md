# USER.md

Use this file for durable facts about the human user.

Capture things like:
- their name and preferred form of address
- timezone and schedule preferences
- priorities, missions, and ongoing projects
- tool and automation preferences
- links to relevant notes in `VAULT/people/`

When you create a person note for the user in `VAULT/people/`, add the link
here immediately.
